[global 폴더 구조 및 역할]

global
├── config
│    - 전역 설정 클래스 관리
│    - 예: WebConfig.java (CORS, 인터셉터, 메시지 컨버터 등)
│    - Spring Boot 앱 전체에서 적용되는 설정
│
├── security
│    - JWT 관련 처리
│    - 예: JwtProvider.java, JwtFilter.java
│    - 토큰 생성/검증/재발급
│
├── exception
│    - 공통 예외 처리
│    - 예: GlobalExceptionHandler.java
│    - 프로젝트 전체에서 발생하는 예외를 통일된 형식으로 처리
│
├── common
│    - 공통 유틸리티 및 상수
│    - 예: DateUtils.java, ResponseMessage.java
│    - 여러 곳에서 재사용 가능한 코드
