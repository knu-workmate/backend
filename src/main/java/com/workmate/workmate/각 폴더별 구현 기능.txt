[WorkMate 기능별 폴더 요약]

1. auth
- 로그인/로그아웃
- JWT 토큰 발급 및 인증
- 토큰 재발급

2. user
- 유저 정보 관리
- 회원가입
- 권한/직급 관리
- 가게 소속 관리

3. work
- 근무 스케줄 관리
- 대타 요청 및 승인
- 근무표 조회/수정

4. communication
- 게시판 생성/조회
- 게시글 작성/수정/삭제
- 댓글 작성/삭제

5. notification
- 알림 생성/조회
- 알림 읽음 처리
- 푸시 알림 발송

6. ai
- LLM 기반 질문 챗봇
- 매뉴얼 문서 DB 저장/검색
- 답변 생성

7. global --> 여러번 재사용 가능한 코드 api화
- JWT 유틸/Provider
  → 토큰 생성, 검증, 만료 확인
- 공통 예외 처리
- 응답 메시지 표준화
- 공통 유틸리티
- 상수 정의

8. pages --> 프론트 엔드에서 요구한 기타 화면 구성을 위한 api 제공
- 메인 화면 구성 위한 사용자 이름, 공지사항 게시글 미리보기 등
  → 기존에 각 기능 폴더에서 정의한 엔티티와 레포지토리를 재사용하기 때문에 controller, dto, service 폴더만 존재.
